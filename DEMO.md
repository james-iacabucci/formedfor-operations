
# Sculptify - Sculpture Management System

## Core Features

### 1. Sculpture Library Management
- Create new sculptures using AI image generation
- Upload existing sculptures
- View sculptures in grid or list view
- Search sculptures by name
- Filter sculptures by:
  - Product line
  - Material
  - Status
  - Height dimensions
  - Tags

### 2. AI-Powered Features
- Generate sculpture images from text prompts
- Generate multiple variations simultaneously
- Control creativity levels for variations
- Automatic generation of:
  - Sculpture names
  - Descriptions
  - Technical specifications

### 3. Product Line Management
- Organize sculptures into product lines
- Assign product line codes
- Quick product line switching via dropdown
- Product line filtering in main view
- Batch operations by product line

### 4. Sculpture Details
- Comprehensive sculpture information:
  - Dimensions (inches & centimeters)
  - Materials and finishes
  - Manufacturing methods
  - Weight specifications
  - Base details
- File management:
  - Multiple image views
  - 3D models
  - Technical drawings
  - Supporting documents
- Image regeneration capabilities
- Variant generation with controlled creativity

### 5. Tag System
- Add and manage tags for sculptures
- Filter sculptures by tags
- Batch tag management
- Tag-based organization

### 6. Documentation & Specifications
- Generate PDF specification sheets
- Multiple PDF formats:
  - No pricing
  - Trade pricing
  - Trade & retail pricing
- Automatic dimension calculations
- Material and finish specifications

### 7. Status Management
- Track sculpture status:
  - Idea
  - Pending
  - Approved
  - Archived
- Status-based filtering
- Visual status indicators

### 8. Base Management
- Separate base specifications:
  - Materials
  - Dimensions
  - Weight
  - Manufacturing methods

### 9. Fabrication Quotes
- Manage fabrication quotes
- Track quote status
- Compare multiple quotes
- Select preferred quotes

## User Interface Features

### 1. Navigation & Views
- Grid view for visual browsing
- List view for detailed information
- Responsive design for all screen sizes
- Quick navigation between sculptures

### 2. Batch Operations
- Select multiple sculptures
- Bulk tag management
- Batch status updates
- Mass regeneration capabilities

### 3. Search & Filters
- Real-time search functionality
- Advanced filtering options
- Combined search and filter capabilities
- Save filter preferences

### 4. Image Management
- High-resolution image display
- Image regeneration
- Multiple view angles
- Bulk image operations

## Getting Started

1. **Creating Your First Sculpture**
   - Click the "Create" button
   - Enter a descriptive prompt
   - Choose creativity level
   - Select product line (optional)
   - Generate variations
   - Save preferred variations

2. **Managing Product Lines**
   - Access Settings
   - Navigate to Product Lines section
   - Create new product lines
   - Assign product line codes

3. **Organizing with Tags**
   - Access Settings
   - Navigate to Tags section
   - Create and manage tags
   - Apply tags to sculptures

## Tips & Tricks

1. **Efficient Browsing**
   - Use keyboard shortcuts for navigation
   - Utilize the search bar for quick access
   - Save commonly used filters

2. **AI Generation**
   - Be specific in prompts for better results
   - Use medium creativity for balanced variations
   - Generate multiple versions simultaneously

3. **File Management**
   - Keep consistent naming conventions
   - Organize files by type
   - Regular cleanup of unused files

## Technical Requirements

- Modern web browser (Chrome, Firefox, Safari)
- Internet connection for AI features
- Sufficient screen resolution (1280x720 minimum recommended)
